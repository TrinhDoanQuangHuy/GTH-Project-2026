# 🌿 Green Tech Hub (GTH)

## Mô tả dự án
Green Tech Hub là nền tảng kết nối các giải pháp công nghệ bảo vệ môi trường. Website giới thiệu dự án (Landing Page) để kêu gọi đối tác.

## Công nghệ sử dụng
- HTML5
- CSS3
- Bootstrap 5
- JavaScript (Vanilla)
- Git (Version Control)

## Cấu trúc dự án
```
green-tech-hub/
├── index.html          # Trang chính Landing Page
├── css/
│   └── style.css       # Custom CSS styles
├── js/
│   └── main.js         # JavaScript logic & form validation
├── images/             # Hình ảnh dự án
├── .gitignore          # Git ignore configuration
└── README.md           # Tài liệu dự án
```

## Quản lý dự án
- **Jira Project**: Green Tech Hub (GTH)
- **Project Type**: Scrum Software Project
- **Workflow**: To Do → In Progress → Code Review → Testing → Done

## Git Flow
- `main` - Nhánh chính (code ổn định)
- `feature/landing-page` - Phát triển giao diện Landing Page
- `feature/contact-form` - Phát triển form liên hệ

## Quy tắc Commit
```
GTH-{issue_number}: {mô tả ngắn gọn}
```
Ví dụ: `GTH-1: add hero section with background image`
